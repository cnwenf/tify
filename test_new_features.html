🚀 AI翻译插件新功能测试

📋 测试说明

请按照以下步骤测试新功能：

  1. 确保已加载最新版本的插件
  2. 打开插件弹窗，查看模型选择是否包含新的选项
  3. 测试不同的翻译模式（双语对照、直接翻译）
  4. 如果有Ollama环境，测试本地模型检测功能
新增 微软翻译服务支持

新增了微软认知服务翻译API支持，提供快速准确的翻译服务。

This is a test sentence for Microsoft Translator service. It should provide fast and accurate translation results.

测试要点：

  • 在模型选择中应该看到"🌐 微软翻译"选项
  • 选择微软翻译时需要配置API密钥
  • 支持多种语言的快速翻译
新增 Ollama本地模型支持

支持本地Ollama服务，自动检测已安装的模型并提供下拉选择。

Hello world! This text can be translated using local Ollama models for better privacy protection.

Ollama设置示例：

# 安装Ollama并下载模型
ollama pull llama2
ollama pull codellama
ollama serve

测试要点：

  • 选择"🦙 Ollama"时应显示模型子选择区域
  • 自动检测本地运行的Ollama模型
  • 支持刷新模型列表功能
  • 无需API密钥，完全本地化
优化 动态模型列表加载

所有AI服务商的模型现在都支持动态加载，不再写死模型ID。

The models are now loaded dynamically with better user experience and descriptions.

改进内容：

  • 模型选择下拉框显示图标和描述
  • 选择不同模型时显示相应提示信息
  • 支持未来扩展更多AI服务商
简化 翻译显示模式

简化翻译模式选择，现在只保留两种最常用的模式。

This paragraph will demonstrate the simplified translation modes. Try both bilingual and direct translation modes.

可用模式：

  • 双语对照（默认）：原文+译文同时显示，完美阅读体验
  • 直接翻译：直接替换原文，鼠标悬停显示原文

移除的模式：

  • 并排对照模式（已移除，简化界面）
优化 默认设置

新的默认设置提供更好的开箱即用体验。

Default settings have been optimized for better out-of-box experience. The bilingual mode is now the default translation mode.

默认配置：

  • 默认翻译模式：双语对照
  • 默认AI模型：OpenAI GPT-3.5
  • 默认目标语言：中文

  3. 翻译模式测试：确认只有两种翻译模式，且默认选中"双语对照"
