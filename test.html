🌐 Tidy Test Page

Use this page to test the AI translation extension functionality

📋 Testing Instructions

  1. Make sure the Tidy extension is installed and enabled
  2. Configure your API key in the extension popup (click the extension icon)
  3. Enable translation in the popup
  4. Try the following test methods:
    • Page Translation: Click "翻译当前页面" in the popup
    • Selection Translation: Select any text below and click the translation button that appears
    • Right-click Menu: Select text and use "翻译选中文本" from right-click menu
    • Keyboard Shortcuts: Alt+A to toggle translation, Alt+W to translate page
    • Float Button: Click the floating button on the right side of the page
